 # Query Execution Plan and SQL Optimization

## Overview
This project focuses on analyzing and optimizing a SQL query execution plan in PostgreSQL. The goal was to understand how PostgreSQL plans a query, interpret the execution cost, and improve query performance by selecting an appropriate index.

## Project Scope
The project included:
- Creating and populating PostgreSQL tables
- Generating SQL test data
- Running a generated SQL query
- Using query execution plans to understand how PostgreSQL processes the query
- Interpreting estimated cost and expected number of returned records
- Comparing query cost after removing predicates
- Drawing a relational algebra representation of the query
- Optimizing the query using one selected index

## Key Concepts Applied
- PostgreSQL query planning
- Query execution plans
- SQL cost estimation
- Predicate analysis
- Relational algebra
- Index selection
- Query optimization
- Performance analysis

## Tools Used
- PostgreSQL
- SQL
- EXPLAIN / query execution plan
- Diagramming tool
