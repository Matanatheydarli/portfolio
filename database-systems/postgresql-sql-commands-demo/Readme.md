# PostgreSQL SQL Commands Demonstration

## Overview
This project demonstrates practical usage of PostgreSQL and core SQL commands through a set of database examples. The goal was to show understanding of relational database concepts, constraints, joins, views, normalization-related table creation, and transaction control.

## Project Scope
The demonstration includes:
- Creating tables with Primary Key and Foreign Key relationships
- Using auto-incrementing values
- Demonstrating referential integrity
- Using ALTER commands to add columns and modify column types
- Applying CHECK, UNIQUE, and NOT NULL constraints
- Demonstrating NATURAL JOIN, INNER JOIN, and OUTER JOIN operations
- Creating regular views and materialized views
- Comparing regular views with materialized views
- Using CREATE TABLE AS SELECT
- Using INSERT INTO ... SELECT
- Demonstrating COMMIT and ROLLBACK transactions

## Key Concepts Applied
- PostgreSQL environment setup and usage
- Relational database design
- Primary keys and foreign keys
- Referential integrity
- SQL constraints
- SQL joins
- Views and materialized views
- Table creation from query results
- Basic normalization concepts
- Transaction control
